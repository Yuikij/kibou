## 【哥飞】：人人都能学会的需求挖掘方法，让你不再只会拍脑袋想需求

**分类**: 关键词与需求挖掘

**一句话总结**: 本文系统讲解了如何通过关键词研究，科学、可持续地挖掘海外用户的真实需求，并以此为基础构建SEO友好的产品。

### 核心要点
- **出海市场潜力巨大**：海外市场拥有约82亿用户，是面向国内14亿用户的6倍；通过地理套利（收美元花人民币），同等努力下收入潜力可达国内市场的42倍。
- **SEO理念核心**：强调“与搜索引擎共建生态”，而非对抗或作弊，这是获取长期稳定免费流量的基础。短期作弊会被算法惩罚。
- **搜索引擎市场份额**：谷歌占据全球约89.66%的搜索市场份额，做SEO主要指做谷歌SEO。
- **挖掘需求即挖掘关键词**：用户需求通过搜索关键词表达。因此，科学挖掘需求就是去寻找有用户搜索的关键词。
- **关键词挖掘工具与方法**：
    - **Google搜索框/相关搜索**：发现未知的关联关键词。
    - **Google Keyword Planner**：需投放少量广告（如每天十几元）获取详细的搜索量等数据。
    - **Google Trends**：查看关键词搜索趋势，判断市场是否在增长（如AI video相关词趋势上升）。
    - **SEO工具**（如文中案例用的SEOQuake的关键词生成器）：输入词根发现大量相关关键词。
    - **研究竞品网站**：顺藤摸瓜，从流量小的网站发现流量更大的网站，并深入研究其功能、页面和关键词策略。
- **SEO内容创作关键**：
    - 网页内容必须围绕目标关键词写作。
    - 一个网页通常只聚焦一个主题。
    - 关键词密度建议控制在3%-5%。
    - 每个网页单词数量建议在800以上，过少（如几十、一两百字）不利于谷歌识别内容主题。
- **技术要点**：
    - **后端渲染**是SEO友好的，因为前端渲染（如Vue框架）在服务器首次返回的HTML中缺乏内容，不利于爬虫抓取。
    - **提交Sitemap**对于Bing等搜索引擎至关重要，因其依赖Sitemap来节省爬虫资源。
    - Bing等搜索引擎对标题长度、描述长度、结构化数据等小细节更在意。
- **增强SEO效果的综合策略**：通过软文、SEM广告、红人营销、用户裂变等方式，可以带来自然外链和品牌词搜索量，有助于获得“座链接”（Sitelinks），提升整体SEO效果。Google Keyword Planner中的数据需要投放小额广告才能获取详细数据。
- **案例拆解**：一个满足“睡眠周期计算”需求的纯前端网站，月访问量达170万，年广告收入预估约67万人民币。它匹配了诸如“sleep calculator”，“what time should I go to sleep”等多个用户搜索关键词。
- **养成习惯，培养“搜商”**：将需求挖掘从“任务”转变为基于好奇心的“日常习惯”。速成方法是重复研究至少100个关键词和1000个网站，以建立对关键词和网站的直觉判断力。

### 可执行动作
- 使用Google搜索框、Google Keyword Planner、Google Trends和SEO工具（如SEOQuake），围绕一个产品方向进行关键词扩展和趋势分析。
- 在谷歌搜索结果和工具中，寻找并点击那些流量似乎不大（如长尾词排名靠前）的网站，研究其功能、页面结构和关键词布局。
- 写网页内容时，确保围绕一个核心关键词展开，并覆盖该主题的相关词汇，内容篇幅建议在800单词以上。
- 为你的网站做好后端渲染，并确保向Bing等搜索引擎提交Sitemap。
- 参考案例网站（sleepcalculator.com），分析其如何用简单工具满足一个特定需求，并思考类似的长尾需求是否可被你的产品满足。

- **Google Trends**: 用于分析关键词的搜索趋势。
- **SEOQuake (关键词生成器)**: 用于输入词根，扩展出大量相关的用户搜索关键词。

## 第一位嘉宾【茄子】：广告投放的策略思考

**分类**: 广告投放
**一句话总结**: 广告投放的核心是算账，通过了解平台智能程度并学会控制出价与预算，可实现快速上手。

### 核心要点
- **算账是前提**：需要计算现金流、回本周期（案例中回本周期曾有一年、半年、一个月至三个月不等），并持续评估产品复购、卖点、弃单挽留、邮件二次触达等因素。自然流量与付费流量的用户价值（客单价、转化率）存在差异。
- **选择投放平台**：需综合考虑平台特性、用户画像及智能程度。Facebook最智能、数据最全但利润空间有限；谷歌基于需求搜索，冷启动久但稳定；TikTok成本波动大、对素材要求高，但存在流量红利；网红（KOL）流量聚合平台，但需具备垃圾流量处理能力。
- **快速上手策略（六十分方案）**：关键在于“学会控制”，即在可控的出价效果内，通过控制投放速度来增加转化效果的可控性。Facebook单个广告系列预算控制在10-80美元内成本较可控，超过80美元后建议复制新系列。
- **TikTok渠道策略**：平台智能化程度不高，可通过“出极致低价”提升成功可能性。但该渠道垃圾流量率高，需借助第三方工具设置规则来屏蔽垃圾网站/流量。
- **广告结构关键要素**：包括账户搭建结构、产品变现能力、预算（决定跑量速度）、出价（与成本正相关）、CPM（千次曝光成本）、素材点击率与转化率（决定跑量潜力）。
- **测试阶段**：先“低预算高出价”测试，让系统认识产品，初始预算可为5-10美元。若市场成本过高（如美国），可先选择西班牙、德国、法国等成本较低的市场测试。
- **优化是循环过程**：每轮优化可重点迭代一个模块，如第一轮迭代产品，第二轮迭代投放流程（使用自动化工具），第三轮迭代投放细节（如测试Facebook Reels视频版位）。
- **放量期**：拓展多国家（如美国、德国、法国、日本）和多语言（建议做西班牙语和葡萄牙语市场，覆盖人群大）。
- **素材制作流程**：使用Adspy工具找到可模仿的竞品案例，将案例投喂给Claude协助拆解脚本、文案并生成提示词，再用Midjourney网站生成类似风格的素材。
- **KOL投放红利**：寻找最近一个月内在平台（如TikTok、YouTube）起量的博主，他们报价相对较低且可协商，重点看其配合度。
- **构建独特竞争力**：需根据团队基因（如产品力、技术力、精细化运营、素材能力、地域成本优势、资源优势等）进行取舍和发力，小团队的优势在于成本低、灵活性高。
- **新客优惠**：Facebook、TikTok、谷歌等平台通常有针对新投放客户的大额优惠券或返点活动，可降低初期投放成本。

### 可执行动作
- 为产品建立详细的成本核算表，包含所有成本模块，并在投放过程中每周、每月根据新理解持续调整。
- 针对不同平台的智能程度，采用不同的控制策略：Facebook控制单系列预算在80美元以内；TikTok尝试出极致低价并使用工具过滤垃圾流量。
- 从美国、德国、法国、西班牙等不同成本市场的测试中积累数据，选择适合的市场作为起点。
- 使用Adspy工具调研竞品广告素材，并利用Claude和Midjourney批量生成广告素材。
- 关注并尝试使用Facebook、TikTok、谷歌等平台针对新客的优惠券或返点活动。
- 对于KOL投放，通过工具寻找近期快速起量的博主进行合作。

### 提到的工具/网站
- **Adspy/Bigspad (广大了)**: 竞品广告素材与案例调研工具。
- **Google Ads Transparency Center**: 谷歌官方公开的广告素材库，可免费查看所有在跑产品的素材和文案。
- **Claude**: 用于批量拆解脚本、文案，翻译文案，生成提示词。

- **广告优化与拓展**: 持续投放，对效果好的广告系列加大预算，效果差的减少。可探索将流量低但转化好的国家/地区加入投放测试。
- **用户运营与产品优化**: 利用邮件营销（发优惠券、召回取消订阅用户）和行为分析工具（如观看用户使用回放、分析用户流失节点）来持续优化产品。

- AI SaaS网站是普通人触手可及的低门槛独立开发方向，上手快。
- 核心优势：能快速获得正反馈（例如提交后当天可能看到流量），变现模式常见（广告、订阅）。
- 产品构思来源：个人想做、复刻成功案例（如蹭VU3热度）、抓住近期热点。
- 标准操作流程（SOP）：想清楚做什么 -> 取一个有品牌词的好记名字 -> 购买域名 -> 快速上站 -> 推广、看数据、运营。
- 快速开发途径：借助AI代码生成工具（如Web Code）或使用模板（开源或商业模板）。
- 推广关键：提交谷歌Search Console (GSC) 以获得搜索流量，并配置Google Analytics (GA) 进行数据分析。
- **ShipAny框架** 核心功能内置：包含蓝鲸配置、用户控制中心、管理后台，以及登录、注册、支付、多语言、存储等基础设施。
- 案例一：通过提供免费GPU服务，利用用户自发传播，在短时间内获得数百万流量。
- 案例二：非技术背景的产品经理使用框架，约一个月时间完成第一个项目，后续开发速度加快。
- 未来规划：将ShipAny打造为“AI版的WordPress”，开发更多业务模板（如图片、视频生成类），降低上线门槛。
- 独立开发者成功关键：重视社交媒体运营，持续输出有价值内容以积累粉丝，利于产品推广。

### 可执行动作
- 为自己的AI产品构思一个独特、易记且不侵犯他人权益的品牌名称。
- 购买一个相关域名。
- 使用ShipAny框架或选择合适的模板进行快速网站开发。
- 网站上线后，立即将其提交至谷歌Search Console (GSC)。
- 为网站添加Google Analytics (GA) 以追踪和分析流量数据。
- 在产品导航站提交链接，或通过软文等方式进行推广。
- 关注并学习ShipAny提供的新模板（如图片、视频生成类），以快速搭建特定功能网站。

### 提到的工具/网站
- **ShipAny**: 用于快速搭建AI SaaS网站的框架/模板，内置用户、支付、多语言等系统。
- **Web Code (指AI代码生成工具)**: 通过自然语言描述快速生成网站代码。
- **GitHub**: 寻找开源或商业化网站模板。
- **VPS (如Cloudflare Pages/Workers)**: 用于部署网站。
- **谷歌Search Console (GSC)**: 提交网站以获得谷歌搜索流量。
- **Google Analytics (GA)**: 网站数据统计与分析工具。
- **Product Hunt**: 发布和推广产品的平台。
- **京东**: 嘉宾新书《MCP》的预售平台。
