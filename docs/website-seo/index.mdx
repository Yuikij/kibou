## 学习路径建议

1. **第一周,跑通闭环**:读[新手入门](./01-新手入门/新手入门.md),照着快速上站 SOP 上线第一个单页工具站,提交 GSC。
2. **第二至四周,练找词**:读[关键词与需求挖掘](./02-关键词与需求挖掘/关键词与需求挖掘.md),每天用固定方法找词、建评估表,选出 KD 低、有真实意图的词。
3. **持续迭代**:按[建站与OnPage](./03-建站与OnPage/建站与OnPage.md)逐项检查页面;收录稳定后按[外链建设](./05-外链建设/外链建设.md)加外链。
4. **有流量后**:接入 [Adsense](./07-变现/变现.md) 或订阅收费;读[案例拆解](./10-案例拆解/案例拆解.md)和[经验复盘](./11-经验复盘/经验复盘.md)找放大思路。
