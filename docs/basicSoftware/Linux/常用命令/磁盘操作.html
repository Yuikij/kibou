文件和目录操作​

ls - 列出目录内容​

ls [选项] [目录]

常用选项：

  • -l: 使用长列表格式
  • -a: 显示所有文件，包括隐藏文件

示例：

ls -la /home/user

cd - 切换目录​

cd [目录]

示例：

cd /var/log

系统信息​

uname - 显示系统信息​

uname [选项]

常用选项：

  • -a: 显示所有信息

示例：

uname -a

用户和权限管理​

chmod - 修改文件权限​

chmod [选项] 模式 文件

示例：

chmod 755 script.sh

进程管理​

ps - 显示进程状态​

ps [选项]

常用选项：

  • -ef: 显示所有进程

示例：

ps -ef | grep nginx

网络相关​

ping - 测试网络连接​

ping [选项] 主机

示例：

ping -c 4 google.com

软件包管理​

apt - Debian/Ubuntu 软件包管理器​

apt [选项] 命令

示例：

sudo apt update
sudo apt install package_name
